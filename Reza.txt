Pour avoir username du current user :
Meteor.user().username

Pour avoir email du current user :
Meteor.user().emails[0].address

ID current user:
Meteor.userId() ou Meteor.user()._id

Pour envoyer email de verification de l'adresse (SUR LE SERVEUR) :
Accounts.sendVerificationEmail(userId)
